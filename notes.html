Introduction to C Programming

C is a powerful general-purpose programming language. It is fast, portable, and available on all platforms.

Basic Syntax

The syntax of C programming is the set of rules that define the combinations of symbols that are considered to be correctly structured programs in that language.

            
#include <stdio.h>

int main() {
    printf("Hello, World!");
    return 0;
}

Data Types

Data types in C refer to an extensive system used for declaring variables or functions of different types.

Primary Data Types

  • int: Used to store integers. Example: int a = 10;
  • float: Used to store floating-point numbers. Example: float b = 3.14;
  • double: Used to store double-precision floating-point numbers. Example: double c = 3.14159;
  • char: Used to store single characters. Example: char d = 'A';

Derived Data Types

  • Arrays: Collection of elements of the same type. Example: int arr[5] = {1, 2, 3, 4, 5};
  • Pointers: Variables that store the address of another variable. Example: int *ptr = &a;
  • Structures: Custom data types that group variables of different types. Example: 
    
struct Person {
    char name[50];
    int age;
    float salary;
};

Enumeration Data Type

enum: Used to define a variable that can hold a set of predefined constants. Example: 


enum Day {
    Sunday,
    Monday,
    Tuesday,
    Wednesday,
    Thursday,
    Friday,
    Saturday
};
enum Day today = Monday;

Void Data Type

void: Represents the absence of type. Commonly used in functions to indicate that they do not return a value. Example: 


void myFunction() {
    // function code
}

Control Structures in C Programming

C programming language provides several control structures to manage the flow of execution. Here is a brief overview:

1. Conditional Statements

  • if - Executes a block of code if its condition is true.
  • if-else - Executes one block of code if the condition is true, and another if it is false.
  • switch - Selects one of many blocks of code to execute based on the value of an expression.

2. Loops

  • for - Repeats a block of code a specified number of times.
  • while - Repeats a block of code as long as a condition is true.
  • do-while - Similar to while, but guarantees that the block of code is executed at least once.

3. Jump Statements

  • break - Exits from the innermost loop or switch statement.
  • continue - Skips the remaining code in the current iteration of a loop and proceeds to the next iteration.
  • return - Exits from a function and optionally returns a value.

These control structures are fundamental for creating efficient and effective C programs.

Functions in C Programming

Functions are a core concept in C programming, used to modularize and reuse code. Here's a brief overview:

1. Definition

A function in C is a block of code designed to perform a specific task. It is defined once and can be called multiple times from different parts of a program.

2. Syntax

The syntax for defining a function in C is:


    return_type function_name(parameters) {
        // body of the function
        return value; // optional, depending on return_type
    }

3. Components

  • return_type - Specifies the type of value the function returns. Use void if no value is returned.
  • function_name - The name used to call the function.
  • parameters - A list of variables passed to the function. Can be empty if no parameters are needed.
  • body - The block of code that performs the function's task.
  • return - A statement that returns a value to the caller. Not used if the return type is void.

4. Example

Here's a simple example of a function that adds two integers:


    #include <stdio.h>
    
    int add(int a, int b) {
        return a + b;
    }
    
    int main() {
        int result = add(5, 3);
        printf("Sum: %d\n", result);
        return 0;
    }

In this example, the add function takes two integers as parameters, adds them, and returns the result. The main function calls add and prints the result.

Functions in C are defined as a block of statements that perform a specific task.
